## 元件規範

### 按鈕

| 變體 | 用途 | 樣式 |
|------|------|------|
| Primary | 主要操作（儲存、提交） | 填色 + Primary 色 |
| Default | 次要操作（取消、返回） | 描邊 |
| Danger | 危險操作（刪除） | 填色 + Danger 色 |
| Text | 輔助操作 | 純文字 |

**尺寸**：Small / Medium（預設）/ Large

**狀態**：default / hover / active / focus / disabled / loading
